Beverage

Because Gulp.js doesn't have that extra slice of lemon!
(nor such bad taste when choosing a logo font)

You know Diet Coke ? Discover Diet Gulp!

Beverage is a port of Gulp.js to PHP.

Why "yet-another-Gulp-clone"?

I personnaly love Gulp, but as a Windows user, some Unix dependecies are a hassle to install.

Plus, I find the way NPM manages dependencie is pure chaos. Node_Modules folder alone represents 80% of the size of most of my projets.

The goal of Beverage is to give the power of Gulp and tis ease of use to PHP projects as a Composer package.

Setup

Use Composer to install

{
    "require-dev": {
        "awakenweb/beverage" : "dev-master"
    }
}

Then follow the instructions of the github repository README

Official modules

  • CSS minifier and combiner Included!

  • JS minifier and combiner Included!

  • Scss and Compass compiler

  • Images processing (optimize, watermark, resize, ...) In development

  • Assets versionning In development

  • Assets reloader In development
